# Prompt Iterations

## Iteration 1

### Prompt
Generate a blog about coffee.

### Result
The output was informative but lacked SEO structure and keyword optimization.

---

## Iteration 2

### Prompt
Generate an SEO-friendly blog for Brew Haven Café targeting coffee lovers.

### Improvements

- Better headings
- Improved readability
- More engaging introduction
- Included relevant keywords

---

## Iteration 3

### Prompt
Act as a professional SEO content writer.

Generate a 1000-word blog including:

- SEO Title
- Meta Description
- H1, H2, H3 Headings
- Primary and Secondary Keywords
- FAQ Section
- Internal Linking Suggestions
- Strong Call-to-Action

### Result

The blog became well-structured, optimized for search engines, and suitable for publication.

---

## Learning

Refining prompts improved:
- SEO quality
- Keyword placement
- Readability
- Content organization
- User engagement
